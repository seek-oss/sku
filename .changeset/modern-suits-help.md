Bumping vite version to vite@8

This release shouldn't impact apps unless incompatible vite@8 options have been added to `dangerouslySetViteConfig`. This API can be susceptible to unexpected changes, so this release is not marked as breaking.

If your app is using `@sku-lib/vite`, please update it to version 2.0.0+.
